# Changelog

本文档记录 llmPylon 项目的所有重要变更。

---

## v1.7.0 (2026-05-24)

### 新增
- 白天/黑夜模式切换（CSS 变量驱动，侧边栏一键切换，持久化到 localStorage）
- 中/英文国际化支持（vue-i18n v9，`locales/zh.json` + `locales/en.json`）
- 应用管理新增网格视图和用量视图（仿厂商管理三视图模式）
- 服务端新增 `/api/keys/usage` 接口，按应用聚合请求数、Token 数、活跃度
- 厂商卡片渐变色条按 URL 域名自动匹配厂商类型（DeepSeek 蓝、OpenAI 绿...未识别按 hash 分配 14 色）

### 样式
- 全面重构 CSS 设计系统：CSS 变量令牌（12 个语义颜色）、发光卡片（glow-card）、毛玻璃侧边栏/顶栏、网格背景 + 径向光晕球
- 统计页面全面改造：KPI 卡片重构为 StatsCard 布局、图表卡片加渐变色顶线、新增分类分割线（SegmentLabel）
- ECharts 图表美化：统一 tooltip 卡片样式、渐变色柱状图、丝滑折线、环状图动态 HSL 配色、热力图改为手写蓝色系方格

### 修复
- 16 个模态框统一样式并居中显示
- 饼图图例移至底部避免覆盖图形
- ECharts tooltip 溢出卡片被裁剪的问题
- 错误率趋势图（`errorRateOption` computed 函数被误删，已恢复）
- 卡片渐变色条溢出圆角边界
- groq 厂商颜色与 anthropic 撞色（改为石灰绿 `#84cc16`）

## v1.6.0 (2026-05-21)

### 新增
- 厂商用量视图（网格/列表/用量三模式），显示日用量柱状图、余额/月消费、模型分布
- 厂商余额/配额 API（DeepSeek、Moonshot、Venice 自动拉取余额；OpenAI / Anthropic 用量缓存）
- 管理员时区设置（所有时间戳按所选时区显示）
- API Key 显示/隐藏切换按钮

### 增强
- 通知系统新增"错误通知"类型，支持错误抑制时间（防止风暴）
- 统计面板 UI 统一化（深色头部转为浅色卡片风格）
- 全局统计按厂商、流式/非流式、协议类型筛选

## v1.4.0 (2026-05-07)

### 新增

- README.md 中英文同步更新

## v1.3.0 (2026-04-23)

### 新增
- 统计模块 v2：Token 入/出/总计精确追踪、流式指标（TTFB/中断率）、延迟百分位（P50/P90/P99）、慢/错请求 Top 10

### 修复

- 流式优化：块超时、TTFB 记录、完成校验、自动重试
- 流式协议转换行缓冲修复 TCP 分包问题
- 协议转换完整支持 tools/tool_choice/tool_use/tool_result
- reasoning_content/thinking 块在转换中保留
