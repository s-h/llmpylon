# Security

## Deployment scope

**This project is intended for trusted networks (e.g. internal lab, personal homelab). Do not expose the admin UI and proxy ports directly to the public Internet.** Default credentials and API keys are high-risk if discoverable.

## Reporting issues

If you believe you found a security vulnerability, **do not** disclose details in a public issue. Prefer:

- GitHub **Private vulnerability reporting** (if enabled on the repository), or
- A private message to maintainers with reproduction and impact

We will respond as soon as we reasonably can.
